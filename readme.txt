1.安装python3.5以上  https://www.python.org/downloads/

2.安装pycharm  http://www.jetbrains.com/pycharm/download/#section=windows

3.安装PyQt5： 命令提示符 pip3 install PyQt5
4.安装pyOpenGL  命令提示符 pip install PyOpenGL PyOpenGL_accelerate
5.安装pyqtgraph   http://www.pyqtgraph.org/

打开项目，file->open 选择 workspace_python 文件夹。然后打开GimbalContrlSystem_v6.py,右键run

3、4、5也可以在pycharm里面按顺序完成：
打开pychram->File->Seting->Project->Project Interpreter->添加库（绿色‘+’号）
